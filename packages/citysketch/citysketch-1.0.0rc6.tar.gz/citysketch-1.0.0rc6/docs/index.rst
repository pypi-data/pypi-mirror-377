Key Features
============

* Interactive Building Creation
* Real-time 3D Visualization
* Basemap Integration
* GeoTIFF Overlay Support
* CityJSON Export
* AUSTAL Integration

Quick Start
===========

1. Launch CitySketch
2. Select a basemap location
3. Add buildings with toolbar buttons
4. Edit properties with number keys
5. View in 3D with F3
6. Save your project
